# Выявление закономерностей, которые определяют успешность игры
Исследование рынка игр: используя данные о продажах игр, оценках, жанрах и платформах, выявила закономерности и определила признаки успешности игр.

Проект был реализован в рамках курсах Яндекс.Практикум по профессии «Аналитик данных».

| :---------------------- | :---------------------- | :---------------------- |
